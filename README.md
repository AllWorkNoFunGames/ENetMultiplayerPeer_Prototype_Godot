ENetMultiplayerPeer_Prototype_Godot | NetworkTest01

A Godot testing project for testing vanilla High-Level-MP-API 

Includes:
- Player characters
- Chat System
- State Syncing
- Sending chat messages into a local API (Deno) via Godot HTTPRequest
- And then saving those messages into a sqlite database
